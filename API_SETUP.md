## ‚ùì Common Questions

### Q: Is the Gemini API really free?

**A:** Yes! The free tier is generous and perfect for student use. You don't need a credit card.

### Q: What if I exceed the free limits?

**A:** You'll get a rate limit error. Just wait a bit or upgrade to paid tier if needed (unlikely for normal use).

### Q: Can I use multiple API keys?

**A:** Yes, but one key per extension instance. You can create multiple keys for different projects.

### Q: Does my API key expire?

**A:** No, it remains valid until you delete it or Google revokes it (usually due to abuse).

### Q: Is my API key secure in the extension?

**A:** Yes! It's stored in Chrome's secure local storage, encrypted by the browser.

## üö® Troubleshooting

### "Invalid API key format"

- Make sure key starts with `AIzaSy`
- No extra spaces before/after
- Copy directly from Google AI Studio

### "API key invalid" error

- Key might be deleted/revoked
- Generate a new one
- Update in extension settings

### "Quota exceeded"

- You've hit daily/minute limits
- Wait and try again later
- Check usage in Google AI Studio

### "Failed to fetch"

- Check internet connection
- Verify API key is correct
- Try regenerating the key
